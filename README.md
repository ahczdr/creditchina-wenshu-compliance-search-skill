# creditchina-wenshu-compliance-search

Codex skill for a two-stage compliance evidence workflow:

1. Download company credit reports from CreditChina.
2. Use the report company/legal-representative names to search China Judgements Online criminal bribery records and save evidence screenshots.

Install by copying the `creditchina-wenshu-compliance-search` folder into:

```text
~/.codex/skills/
```
